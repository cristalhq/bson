# bson

[![build-img]][build-url]
[![pkg-img]][pkg-url]
[![version-img]][version-url]

Package implements BSON encoding and decoding in Go. See https://bsonspec.org/spec.html

## Features

* Simple API.
* Dependency-free.
* Optimized for speed.
* Clean and tested code.

See [docs][pkg-url] for more details.

## Install

Go version 1.18+

```
go get github.com/cristalhq/bson
```
